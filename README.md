# vizgres

A fast, keyboard-driven PostgreSQL client for the terminal.

## Status

Early development. See [docs/](./docs/) for design documentation.

## Planned Features

- **Tree Browser**: Navigate schemas, tables, views, functions
- **Query Editor**: Multi-line SQL editing with syntax highlighting
- **SQL Fixer**: Auto-correct typos before execution (SELEC → SELECT, usres → users)
- **Results Viewer**: Scrollable table with JSONB inspection
- **Schema Autocomplete**: Context-aware suggestions

See [docs/09-roadmap.md](./docs/09-roadmap.md) for the full implementation plan.
